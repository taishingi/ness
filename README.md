# ness

# Dependencies

* MariaDb
* openssl
* cargo 
* Alsa libs
* pocketsphinx
* gcc


# Environment

$NESS_USERNAME   Username 

$NESS_DBNAME     Database's name

$NESS_PASSWORD   Database password


### Indexes Music directory content

```bash
ness --save-albums
```
### Playing music


```bash
ness --listen [name]
```
